# Full code for the paper "Evidence-Grounded Multimodal Mining of Fine-Grained Research Problems and Methods from Computer Vision Papers"

## Dataset
The dataset used in this paper is available at https://pan.baidu.com/s/1iW_PJn8NwLIb4GZK9kZLCg?pwd=i8nw  Just replace the "data" folder.

## Run Code
1.Save your Qwen-VL API key to the qwen_key.txt file.  

2.`python scripts/batch_process_papers.py --root-dir data/output --skip-existing --sleep 2` 
The script reads all the papers in the data/output directory and processes them iteratively. The results generated by our method will be saved in the data/output folder, with each paper stored in its corresponding subfolder.
